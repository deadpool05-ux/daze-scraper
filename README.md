# Daze Signal Engine

Automated Client Acquisition & High-Intent Scraping for Dazeco.

![Daze Signal Engine](public/dazelogo.png)

## Quick Start
1. `npm install`
2. Create `.env.local` with `GEMINI_API_KEY` and `GROQ_API_KEY`.
3. `npm run dev`
